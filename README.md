# Evaluaci-n-en-Contacto-con-el-Docente
 Sistema de Gestión de Libros electrónicos

Integrante
MILTON IZA
Objetivo
Desarrollar un sistema en Go que permita gestionar libros electrónicos mediante servicios web, aplicando buenas prácticas como encapsulación, interfaces y manejo de errores.
Funcionalidades principales
* Agregar libros
* Listar libros
* Buscar libros
* Eliminar libros
* Actualizar libros
* API REST
* Conexión a base de datos
Tecnologías utilizadas
* Go (Golang)
* JSON (serialización)
* HTTP (Servicios web)
* MySQL / PostgreSQL
* GitHub
Fecha
Junio 2026
Ejecución
go run main.go
Proyección futura
Se plantea integrar inteligencia artificial para recomendaciones de libros y desplegar el sistema en la nube con microservicios.
